# Bem vindo a minha sessão de estudos
Repositório criado para aglomerar links de acesso ao repositório e pagina web referente aos meus estudos de front-end

## Introdução
Após 3 anos trabalhando como desenvolvedor fullstack em tecnologias como C++ / SQL / UniGUI / JavaScript resolvi que retornarei meus estudos na área que sempre fui apaixonado, o front-end.
Gostaria de destacar que nessa jornada de retorno irei voltar desde o basico HTML/CSS e caminhar até o avançado React/TypeScript/Next/GraphQL entre outros. 
Espero que o senhor ou senhora que está lendo este README, goste do que irá ver e consiga sentir minha paixão nos meus projetos.

## Objetivos diretos
  1. Desenvolver boas práticas de desenvolvimento (Clean Code, Design Pattern ...).
  2. Desenvolver aplicações web que me force a estudar algo novo sempre.
  3. Descrever no próprio código minhas dificuldades para que quem ler possa se identificar e entender que está tudo bem ter problemas no desenvolvimento e que o importante é tentar até conseguir.
  4. Aperfeiçoar meu algoritmo para compenentizar estruturas afim de tornar projetos simples em projetos escalonáveis.
  5. Aplicar SEMPRE acessibilidade nos projetos afim de tornar inclusivo todo e qualquer aplicação desenvolvida nessa jornada.
  6. Aprender e praticar para me tornar um profissional coeso e eficiente.
